# Setup & Customization Guide

## Prerequisites

Before starting:
- [ ] VS Code or Cursor installed
- [ ] Claude Code extension installed
- [ ] Node.js 18+ installed
- [ ] Interpretive Orchestration plugin installed

### Installing the Plugin

If you haven't installed the plugin yet:

1. Open terminal in VS Code (`Cmd+J` / `Ctrl+J`)
2. Start Claude Code: `claude`
3. Install the plugin:
   ```
   /plugin install linxule/interpretive-orchestration
   ```
4. Verify: `/qual-check-setup`

**Plugin repo:** [github.com/linxule/interpretive-orchestration](https://github.com/linxule/interpretive-orchestration)

## Customizing Your Environment

### Change Theme
1. Press `Cmd+K Cmd+T` (Mac) or `Ctrl+K Ctrl+T` (Windows)
2. Select your preferred theme
3. For dark mode: "Default Dark Modern" or "One Dark Pro"

### Change Font Size
1. Press `Cmd+,` (Mac) or `Ctrl+,` (Windows)
2. Search "font size"
3. Adjust to your preference

### Reset to Defaults
Delete `.vscode/settings.json` and reopen the workspace file.

## Keyboard Shortcuts

| Action | Mac | Windows |
|--------|-----|---------|
| Preview markdown | `Cmd+Shift+V` | `Ctrl+Shift+V` |
| Side-by-side preview | `Cmd+K V` | `Ctrl+K V` |
| Toggle terminal | `Cmd+J` | `Ctrl+J` |
| Toggle sidebar | `Cmd+B` | `Ctrl+B` |
| Command palette | `Cmd+Shift+P` | `Ctrl+Shift+P` |

## Screen Sharing Tips

For workshops and demonstrations:
- Increase font size: `Cmd+` (Mac) / `Ctrl+` (Windows)
- Hide sidebar: `Cmd+B`
- Full screen: `Ctrl+Cmd+F` (Mac) / `F11` (Windows)
